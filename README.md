# Classification-du-risque-de-cr-dit
Projet visant à prédire le niveau de risque (faible ou élevé) d’accorder un crédit à un client à partir de modèles de machine learning développés. Le système exploite les données clients pour analyser les profils, évaluer la solvabilité et aider à la prise de décision en matière d’octroi de crédit.
